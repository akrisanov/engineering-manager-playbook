# Системный аналитик

#### Генеральная функция

Регулярное своевременное обеспечение производственной команды продукта корректно детализированными архитектурными и техническими требованиями к реализуемым задачам, полученным на основе анализа бизнес-требований и пользовательских сценариев.

#### Основные функции

* Анализ функциональных требований, приходящих от бизнеса / стейкхолдеров
* Оценка изменений и их влияния на существующую функциональность платформы и внутренних сервисов, необходимых для ведения операционной деятельности бизнеса
* Проработка пользовательских и системных сценариев:
  * Определение связей между компонентами системы
  * Описание модели данных, контракта взаимодействия между подсистемами: потоки данных, межсерверное взаимодействие, API и т.д.
  * Прототипирование, в том числе с привлечением команды разработки
* Участие в декомпозиции комплексных задач и требований для их эффективной реализации командой разработки своего домена
* Исследование ранее неизвестных сервисов, практик и инструментов для последующего внедрения в процессы команды
* Поддержка требований и технической документации в актуальном состоянии
* Участие в обучении новых сотрудников
