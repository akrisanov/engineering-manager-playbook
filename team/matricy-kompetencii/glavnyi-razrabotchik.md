# Главный разработчик

| **Название грейда**                       | Главный инженер                                                                                                              |
| ----------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| **Для повышения необходимо**              | Показать несколько успешно завершенных проектов, в которых была определена стратегия развития и достигнуты поставленные цели |
| **Основная цель**                         | Запускать масштабные продукты в рамках компании                                                                              |
| **Умение планировать свою работу на**     | 6-12 месяцев                                                                                                                 |
| **Решаемые задачи**                       | На стыке архитектуры, потребностей продукта и бизнеса, а также нескольких инженерных команд                                  |
| **Сложность**                             | Множество конфликтующих приоритетов                                                                                          |
| **Управление рисками**                    | Управление рисками на уровне компании. Разработка архитектуры и практик для снижения рисков.                                 |
| **Необходимость ревью выполненных задач** | Нет                                                                                                                          |
| **Экспертность в домене**                 | Полное понимание доменных областей компании и продукта                                                                       |
| **Понимание продукта**                    | Поддрежка продуктовой стратегии                                                                                              |
| **Влияние на людей**                      | Огранизация                                                                                                                  |
| **Коммуникация**                          | Кросс-командная, стекхолдеры, топ-менеджмент                                                                                 |
| **Вклад в команду**                       | Техническое лидерство, умение демонстрировать достижения инженерной организации, найм                                        |
| **Оценка перформанса**                    | C-level менеджмент                                                                                                           |
| **Опыт продуктовой разработки**           | > 10 лет                                                                                                                     |
